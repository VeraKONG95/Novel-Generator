# Word MCP Server

This local MCP server adds Microsoft Word `.docx` generation and reading for Codex.

## Tools

- `create_word_document`: Build a `.docx` file from sections, paragraphs, and bullets.
- `convert_markdown_to_word`: Convert Markdown into a `.docx` file.
- `read_word_document`: Extract plain text from an existing `.docx` file.

## Local development

Install dependencies:

```bash
npm install
```

Run the server:

```bash
npm start
```
