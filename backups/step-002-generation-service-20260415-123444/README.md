# Noval

一个面向长篇创作流程的桌面小说生成器 MVP。

当前版本包含：

- 项目新建、打开、保存
- 小说设定页
- 蓝图生成页
- 写作页三栏布局
- 章节生成与续写的本地模板逻辑
- 记忆库展示
- Markdown / TXT 导出
- 本地模型设置保存

## 启动

先安装依赖：

```bash
npm install
```

再启动应用：

```bash
npm start
```

## 当前实现说明

- 桌面层使用 `Electron`
- 前端使用原生 `HTML/CSS/JS`
- 项目文件保存为本地 `JSON`
- 如果没有配置真实模型，蓝图和章节会使用内置模板逻辑生成，方便先验证产品形态
